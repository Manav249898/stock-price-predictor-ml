# 📈 Stock Price Predictor (Machine Learning Project)

## 🔹 Overview

This project is a Machine Learning model that predicts stock prices using historical data. It uses Linear Regression to forecast future stock values.

---

## 🔹 Features

* Uses real stock data from Yahoo Finance
* Predicts future stock prices
* Visualizes stock trends using graphs

---

## 🔹 Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-learn
* yFinance

---

## 🔹 How It Works

1. Collects historical stock data
2. Preprocesses the data
3. Applies Linear Regression
4. Predicts future prices

---

## 🔹 Installation

Install required libraries:

```bash
pip install pandas numpy matplotlib scikit-learn yfinance
```

---

## 🔹 How to Run

```bash
python stock.py
```

---

## 🔹 Output

* Predicted stock price
* Graph showing stock trends
